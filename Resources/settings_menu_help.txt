----- Settings Menu Help ------

Enter setting name or ID number to adjust value. 'View' command shows
setting values and their ID numbers.

Note: Do not include backlashes (\) before spaces in directory paths.
Note: Strings must be surrounded by quote marks ("). 

Commands:
    Exit: Exit help menu
    Help: Show help menu
    Print_codes: Print the verbal codes to adjust fields
    Save: Force save program settings (all settings automaticly save when CLC closes).
    View: Display all settings and their values
